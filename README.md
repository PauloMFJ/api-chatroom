# [Chatroom](https://app-ng-chatroom.herokuapp.com/)

This repository contains the api source code for [chatroom](https://app-chatroom-ng.herokuapp.com/).

## Build Setup

``` bash
# First, you run
dotnet restore

# Then to install dependencies
dotnet build

# To compile
dotnet run
```
